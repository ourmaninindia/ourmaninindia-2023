Tutorial: Netlify

Netlify

Netlify

Set up all, or at least both, production and development as aan active branch on Netlify.

Forms

Add data-netlify="true" to the

tag. Also add the url of the success page to the form page.

action="{{ $.Page.Site.Params.form_action }}"

and add form_action = "/success/" to the config.toml page.

So create a new content folder called success with a file called index.md.

Add this immediately below the form tag:

<input type="hidden" name="form-name" value="contact" />

Set canonifyURLs to false in the Hugo configuration file.

Honeypot

Add a netlify-honeypot="bot-field" attribute to the tag and add the following to the form:

<p class="hidden">
  <label>
    Don’t fill this out if you’re human: <input name="bot-field" />
  </label>
</p>

Captcha

There is also an option to add a captcha. Add a data-netlify-recaptcha="true" attribute to your tag. In the place where in the form you’d like the CAPTCHA to appear, add an empty

element with the same data-netlify-recaptcha="true" attribute:

<div data-netlify-recaptcha="true"></div
